\subsection*{備品業務方針}
\writtenBy{\soumuStaff}{奥川}{莞多}
備品業務方針は2019年度後期の方針を継続し，以下の3点とする．
\begin{itemize}
    \item 延滞者への通知
    \item 備品のタグつけ
    \item 私物の明確化
\end{itemize}
延滞者には1週間に1度Slackでの通知を行い，早期返却を促す．2019年度に新たに決めた高額備品の貸し出しについて，実際に貸し出しを行う．問題が発生すれば，その都度問題点を改善していくこととする．私物においては，私物専用の棚におくことを基本とし，長期間の放置は早期撤去を促す．
